Web Application Side

What we need for Web Application side ?

Kindergarten Management Software(KMS) used the following directory structure 

│   app.py
│   db.json
│   config.py
│   data.py
│   setup.py
└───KMS
       │   templates

you can see the directory structure of Kindergarten Management Software(KMS) in following command 

https://github.com/BillelHip/KMSUI

Kindergarten Management Software(KMS) all code in Github we should bring all directory structure to server using command below:

First Installation: The git version control system is installed with the following command 

$ sudo apt install git
Then bring all directory structure of Kindergarten Management Software(KMS) from GitHub 
$ sudo git clone https://github.com/BillelHip/KMSUI.git

You should know TinyDB how it work in Kindergarten Management Software(KMS) link below getting started to TinyDB : 

http://tinydb.readthedocs.io/en/latest/getting-started.html
